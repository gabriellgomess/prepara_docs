ChromeDriver Incompatível

As versões do Chrome e ChromeDriver não são compatíveis. Para corrigir este problema:

  1. Acesse o link: Chrome for Testing
  2. Baixe a versão do ChromeDriver
  3. Substitua o arquivo chromedriver.exe na pasta do projeto
  4. Execute a verificação novamente

Importante: Sempre use a versão do ChromeDriver que corresponde à versão principal do seu Chrome.

Tutorial de Uso

Guia completo do Sistema de Prestação de Contas

Bem-vindo ao Sistema!

Este sistema automatiza o processo de prestação de contas, facilitando o upload e processamento de documentos. Siga os passos abaixo para usar o sistema corretamente.

1

Upload de Arquivos

Planilha Excel

Faça upload da planilha principal com os dados da prestação de contas (.xlsx ou .xls)

Arquivos PDF

Adicione todos os documentos PDF relacionados (notas fiscais, comprovantes, etc.)

Importante: Certifique-se de que todos os arquivos estão corretos antes de prosseguir.

2

Verificação do ChromeDriver

Clique em "Verificar" para confirmar se o ChromeDriver é compatível com seu navegador Chrome.

Se compatível:

Prossiga para as configurações

Se incompatível:

Siga as instruções para atualizar

3

Configurações do Sistema

Modo Headless

Ativado: Executa em segundo plano (recomendado)
Desativado: Mostra o navegador durante a execução

Modo Simulação

Ativado: Testa sem enviar dados reais
Desativado: Execução normal com envio de dados

4

Preenchimento de Dados

Datas

  • • Data de Emissão: Data dos documentos
  • • Data de Pagamento: Data de quitação

Órgão e Parceria

  • • Selecione o órgão responsável
  • • Escolha a parceria correspondente
5

Execução do Robô

Após configurar tudo, clique em "Iniciar Robô" para começar o processamento automático.

Terminal de Execução

Acompanhe o progresso em tempo real através do terminal na parte inferior da tela.

Dicas Importantes

  • Mantenha o navegador Chrome atualizado
  • Verifique a conexão com a internet antes de iniciar
  • Use o modo simulação para testar antes da execução final
  • Não feche o sistema durante a execução do robô
